# C Programming Projects

A collection of beginner-friendly C programming projects.

## Projects Included

### 1. Number Guessing Game

A game where you try to guess a random number. The program gives hints.

**How to Run:**
```bash
gcc -o number_guessing number_guessing.c
./number_guessing
```

**Concepts:**
- Random numbers
- Loops (while)
- If-else conditions
- User input/output

**Example:**
Guess the number between 1 and 100
Guess: 50
Your guess is too high
Guess: 25
Your guess is too low
Guess: 37
yes you got it right.it is 37
no. of attempts = 3

### 2. Snake Water Gun Game

A two-player game (you vs computer).

**How to Run:**
```bash
gcc -o snake_water_gun snake_water_gun.c
./snake_water_gun
```

**Game Rules:**
- Snake drinks water (Snake beats Water)
- Water drowns gun (Water beats Gun)  
- Gun shoots snake (Gun beats Snake)

**Concepts:**
- Random number generation
- Conditional logic
- Game loops
- Score tracking

## Skills Learned

- C fundamentals (variables, loops, functions)
- Conditional logic (if-else)
- Random numbers with rand()
- User input validation
- Game logic
